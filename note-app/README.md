# Priority Note taking app

Just a simple note taking app where the user can:

1. Write a title
2. Assign priority
3. Select a category
4. Write a description
5. Add a note (create a list)
6. Delete a note

This demonstrates use of forms, controlled inputs, hooks, and conditional rendering.
